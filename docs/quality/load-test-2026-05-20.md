# Voice capacity load test — 2026-05-20

## Scope

H5 launch-quality bar: establish a repeatable voice Media Streams capacity run and record provenance in `packages/api/.launch-quality-acks.json`.

## Harness (local)

```bash
cd packages/api
npm run voice-load:selfcheck
```

This exercises `scripts/voice-load-selfcheck.ts` against a local mock WebSocket server and stamps `voice_capacity_run` with provenance `harness-self-check`.

## Staging run (required before high-traffic launch)

```bash
cd packages/api
VOICE_LOAD_TEST_URL=wss://<staging-host>/api/telephony/stream \
VOICE_LOAD_TEST_TARGET=50 \
npm run voice-load:staging
```
